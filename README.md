## 🍀 프로젝트 소개

**부동산 매물 실시간 가격 조회 및 주변 분석을 통해 가장 적합한 매물을 찾아주는 서비스**

#### 컨셉
- Nestquick은 Nest(둥지) + quick(빠르다)의 합성어
- 고객이 원하는 집을 빠르게 찾아준다는 의미 내포

#### 강점
- 빠르고 간단하게 사용자가 원하는 정보 제시
- 간편한 매물 상세정보 조회
